Zenius is a full-stack web platform designed to help students discover internships, hackathons, and career opportunities in one place. It provides an easy-to-use interface for browsing opportunities and a structured backend for managing data.
Features
View and search internships
Browse hackathons
Add new internship entries (admin functionality)
Filter and search functionality
Responsive UI with modern design
Backend API integration



Tech Stack
Frontend:
React.js
Tailwind CSS
Axios

Backend:
Node.js
Express.js
MongoDB
Mongoose

Deployment:
Vercel (Frontend)
Backend deployed separately



Purpose
Zenius aims to simplify the process of finding internships and hackathons for students. It serves as a centralized platform for opportunities and helps students stay updated with career-building events.
